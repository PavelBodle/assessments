# Resolution Note — Print jobs stuck in queue

## Summary
Ticket TCK-00023 (P2) reported by USR-00207 in Engineering: "Print jobs stuck in queue".
The 3rd floor printer shows 'offline' and my print jobs just pile up in the queue.

## Affected Scope & Environment
Affected user USR-00207 (Engineering); system/asset: Printer.
No related tickets; appears isolated.

## Diagnosis & Root Cause
Print spooler service hung on the print server, holding queued jobs and reporting the device offline.

## Resolution / Recommended Fix
Clear the stuck queue and restart the Print Spooler service on the print server; re-add the printer by IP if it stays offline.

## Verification
Confirmed with the user that the issue no longer reproduces after the fix; monitored for 24h with no recurrence.

## Preventive Action / KB Update
Add a proactive check and user guidance to prevent recurrence. See KB-004 for the step-by-step guide.
