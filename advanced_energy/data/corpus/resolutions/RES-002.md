# Resolution Note — Need access to CRM Sales module

## Summary
Ticket TCK-00007 (P1) reported by USR-00220 in Sales: "Need access to CRM Sales module".
I moved to the Sales team and need the CRM Sales module added to my account.

## Affected Scope & Environment
Affected user USR-00220 (Sales); system/asset: CRM.
Related/duplicate tickets exist for this recurring problem.

## Diagnosis & Root Cause
A stale cached credential on the user's mobile device (Exchange ActiveSync) kept retrying the old password, tripping the AD lockout policy.

## Resolution / Recommended Fix
Unlock the AD account, then remove and re-add the corporate mail account on the user's phone so it caches the new password. Verify no mapped drives use old creds.

## Verification
Confirmed with the user that the issue no longer reproduces after the fix; monitored for 24h with no recurrence.

## Preventive Action / KB Update
Add a proactive check and user guidance to prevent recurrence. See KB-003 for the step-by-step guide.
