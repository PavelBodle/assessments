# Resolution Note — Outlook not syncing new mail

## Summary
Ticket TCK-00005 (P2) reported by USR-00138 in Ops: "Outlook not syncing new mail".
Outlook shows 'Disconnected' and no new mail has arrived since this morning, though webmail works fine.

## Affected Scope & Environment
Affected user USR-00138 (Ops); system/asset: Outlook.
Related/duplicate tickets exist for this recurring problem.

## Diagnosis & Root Cause
Corrupted local OST cache after an interrupted sync, leaving Outlook unable to reconcile with the Exchange mailbox.

## Resolution / Recommended Fix
Close Outlook, rename the .ost file to force a fresh download, then restart. If it recurs, recreate the Outlook profile.

## Verification
Confirmed with the user that the issue no longer reproduces after the fix; monitored for 24h with no recurrence.

## Preventive Action / KB Update
Add a proactive check and user guidance to prevent recurrence. See KB-002 for the step-by-step guide.
